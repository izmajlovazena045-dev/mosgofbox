# MOSGOFBOX

Этот бот предназначен для автоматизированного расчета стоимости производства картонных коробок на основе заданных размеров и параметров материала. Результаты расчетов и действия пользователей сохраняются в Google Таблицы.

## Функционал
*   **Диалоговое меню:** пошаговый опрос пользователя для сбора данных.
*   **Расчеты:** вычисление стоимости коробки, стоимости штанцформы и минимальной партии.
*   **Интеграция с Google Sheets:** логирование действий пользователей и их данных в облачную таблицу.
*   **FSM (Finite State Machine):** использование состояний для корректного ведения диалога.

## Технологический стек
*   Python 3.x
*   `aiogram` (версия 3.x) — для работы с Telegram Bot API
*   `google-api-python-client` — для взаимодействия с Google Sheets
*   `asyncio` — для асинхронной работы

## Настройка перед запуском

1.  **Токен бота:** В файле `main.py` замените значение переменной `BOT_TOKEN` на токен, полученный у [@BotFather](https://t.me/botfather).
2.  **Google Sheets API:**
    *   Создайте проект в [Google Cloud Console](https://console.cloud.google.com/).
    *   Включите **Google Sheets API** и **Google Drive API**.
    *   Создайте сервисный аккаунт и скачайте JSON-ключ.
    *   Переименуйте файл ключа в `credentials.json` и поместите его в папку с проектом.
    *   Поделитесь доступом к вашей Google Таблице с `client_email` сервисного аккаунта (дайте права редактора).
3.  **ID таблицы:** Вставьте ID вашей таблицы в переменную `SPREADSHEET_ID` в `main.py`. (ID находится в URL таблицы: `docs.google.com/spreadsheets/d/ID_ЗДЕСЬ/edit`).

## Установка зависимостей

Установите необходимые библиотеки с помощью pip:

```bash
pip install aiogram google-api-python-client google-auth
```

## Запуск бота

Выполните команду в терминале:

```bash
python main.py
```

## Логика расчетов
Бот запрашивает:
1.  **Размеры коробки** (Ширина, Высота, Глубина).
2.  **Код материала** (или использует значения по умолчанию).
3.  **Тип конструкции** (самосборная или под штанцформу).

На основе этих данных бот использует математические формулы для расчета площади материала, стоимости вырубки и количества съемов штанцформы.
